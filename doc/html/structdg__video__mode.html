Detailed Description

Define the size and pixels of the video.

The video mode define the size of the window in pixel, and the number of bits per pixels (number of colours per pixels). if you don't know what to use, choose 32 bpp.

Parameters
width(ussigned int) in pixel.
height(ussigned int) in pixel.
bpp(ussigned int) bit per pixel (32 by default).
